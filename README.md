# Golang Mini E-Commerce GRPC Microservices with Multiple Container

### Running
  - Need Docker and Docker Compose on your Machine
  - Run Command

    ```shell
    $ make up
    ```
  - Stop Container

    ```shell
    $ make down
    ```
